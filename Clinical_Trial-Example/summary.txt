modelo sequencial - é uma pilha linear de camadas, onde cada camada tem apenas um input e output, sendo que o input de uma é o output de outra. Isto é ao contrário de um modelo com pilhas de camadas não lineares, onde as camadas podem ter multiplos inputs e outputs.
